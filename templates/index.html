Timestamp Microservice

User stories:

  1. 1) I can pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date (example: January 1, 2016).

  2. 2) If it does, it returns both the Unix timestamp and the natural language form of that date.

  3. 3) If it does not contain a date or Unix timestamp, it returns null for those properties.

Example usage:

https://freecodecamp-api-project-andydlindsay.c9users.io/February%2014,%201981

https://freecodecamp-api-project-andydlindsay.c9users.io/350956800

Example output:

{ "unix": 350956800, "natural": "February 14, 1981" }
